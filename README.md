# Business Sales Dashboard (Power BI & MySQL)

## Project Overview
This project demonstrates an end-to-end business sales dashboard created using Power BI and MySQL.

## Tools & Technologies
- Power BI
- MySQL
- SQL

## Key Features
- KPI cards for Total Sales, Total Profit, and Quantity Sold
- Sales analysis by region, segment, and category
- Interactive filters and slicers
- Clean and professional dashboard layout

## Data Source
Sales data stored in MySQL and connected to Power BI for visualization.
